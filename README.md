# roshanis.github.io

This repository contains the source for my personal site and digital garden at [roshanis.github.io](https://roshanis.github.io).

## What is here

- A Jekyll-based home page
- Post layouts for essays and notes
- A small collection of writing under `_posts/`

## Local preview

If you already have a Jekyll workflow set up, use your normal local preview flow. This repo also includes `preview.html` for lightweight inspection of the current design.

## Structure

- `index.html`: home page content
- `_layouts/`: page and post layouts
- `_posts/`: dated writing entries
- `_config.yml`: Jekyll configuration

## Status

Current status: active personal writing site.
